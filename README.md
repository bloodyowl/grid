# grid

stylus grid

* em-based
* rtl-friendly

## install

```sh
$ npm install bloody-grid
```

## usage

```stylus
@require "bloody-grid"
```

## structure

```html
<div class="Grid">
  <div class="Grid-col Grid-col--3">
    foo
  </div>
  <div class="Grid-col Grid-col--3 Grid-col--prefix-3">
    bar
  </div>
</div>
```
